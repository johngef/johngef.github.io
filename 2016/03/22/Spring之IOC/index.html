Spring之IOC

发表于 2016-03-22   |   分类于 pring

首先说一下什么是IOC，英文为Inversion of Control，翻译成中文就是控制反转，当然有的人为了更好的理解IOC，把它称作为依赖注入，对IOC更详细的说明看一下百度百科就行了，下面主要说一下IOC的应用：

  • 装配一个bean
  • 参数注入
  • 方法注入
  • bean之间的关系
  • bean作用范围

装配一个bean

首先是配置一下spring环境，现在的spring版本已经来到了4了，这是jar下载地址，下载之后呢，你会看到非常多的jar包，但核心包不多，如下图：

spring文件夹的jar就是这些核心包，这里呢，我也用到了junit，所以也引进了这个jar包。
然后说一下怎么样装配一个bean：
首先我们创建一个class文件，叫做Student:

public class Student {
    private int id;
    private String name;
    private int age;
    private Teacher teacher;

    public Teacher getTeacher() {
        return teacher;
    }

    public void setTeacher(Teacher teacher) {
        this.teacher = teacher;
    }
    public int getId() {
        return id;
    }
    public void setId(int id) {
        this.id = id;
    }
    public String getName() {
        return name;
    }
    public void setName(String name) {
        this.name = name;
    }
    public int getAge() {
        return age;
    }
    public void setAge(int age) {
        this.age = age;
    }

    @Override
    public String toString() {
        return "Student [id=" + id + ", name=" + name + ", age=" + age + ", teacher=" + teacher + "]";
    }

}

然后配置spring的XML文件：

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"  
         xmlns="http://www.springframework.org/schema/beans"  
         xsi:schemaLocation="http://www.springframework.org/schema/beans 
         http://www.springframework.org/schema/beans/spring-beans-3.0.xsd"
         > 
      <bean id = "student" class="com.gefufeng.bean.Student">
          <property name="id" value = "1"></property>
          <property name="name" value = "xiaoge"></property>
      </bean>
</beans>

然后再创建一个junit test case：

public class T {

    private ApplicationContext applicationContext;


    @Before
    public void setUp() throws Exception {
        applicationContext = new ClassPathXmlApplicationContext("spring_config.xml");
    }


    @Test
    public void test() {
        Student student = (Student) applicationContext.getBean("student");
        System.out.println(student);
    }

}
